# Features:
* Includes 3 programmable LEDs to learn about outputs
    * Red `(GPIO - A0)`
    * Yellow `(GPIO - A1)`
    * Green `(GPIO - A2)`
* Includes 1 active buzzer connected to a PWM pin to make melodies
    * Buzzer `(GPIO - D6)`
* Includes 1 push button to learn about input
    * Push Button `(GPIO - D7)`
